Stručný návod na nastavení Fail2Ban

Nastavení ochrany FaiL2Ban pro SSH.

1. Instalace Fail2Ban

Nejprve je třeba Fail2Ban nainstalovat pomocí správce balíčků:

Pro distribuce založené na Debianu/Ubuntu:

sudo apt update
sudo apt install fail2ban

Pro distribuce založené na RHEL/CentOS:

sudo yum install epel-release
sudo yum install fail2ban

Pro distribuce založené na Arch Linuxu:

sudo pacman -S fail2ban

2. Základní konfigurace

Po instalaci je potřeba nastavit konfiguraci Fail2Ban. Originální konfigurační soubor je umístěn ve složce /etc/fail2ban/jail.conf, ale aby nedošlo k jeho přepsání při aktualizaci, vytvořte si vlastní kopii:

sudo cp /etc/fail2ban/jail.conf /etc/fail2ban/jail.local

3. Nastavení služeb (jails)

Otevřete konfigurační soubor /etc/fail2ban/jail.local ve vašem textovém editoru, například:

sudo nano /etc/fail2ban/jail.local

Zde můžete definovat, které služby chcete chránit. Například, pokud chcete chránit SSH, najděte sekci [sshd] a upravte ji podle potřeby:

[sshd]
enabled = true
port = ssh
filter = sshd
logpath = /var/log/auth.log
maxretry = 5
bantime = 3600

4. Spuštění a aktivace Fail2Ban

Po konfiguraci služeb je třeba Fail2Ban spustit a zajistit, aby běžel automaticky po restartu systému:

sudo systemctl start fail2ban
sudo systemctl enable fail2ban

A máme hotovo! Vaše SSH je nyní chráněno před útoky hrubou silou.


Instalace VirtualBoxu na Arch Linuxu

Návod na instalaci prostředí pro virtualizaci.

1. Aktualizace systému

Nejprve je třeba aktualizovat náš systém, aby byly všechny balíčky aktuální:

sudo pacman -Syu

2. Instalace VirtualBoxu

Nyní se můžeme vrhnout na instalaci samotného VirtualBoxu. Instalace je jednoduchá a stačí k tomu jeden příkaz:

Pokud používáte kernel linux (defaultní), příkaz bude:

sudo pacman -S virtualbox virtualbox-host-modules-arch

V případě, že používáte jiný kernel než linux, například linux-lts, bude příkaz pro instalaci VirtualBoxu vypadat následovně:

sudo pacman -S virtualbox virtualbox-host-dkms

3. Načtení kernel modulů

Nakonec je zapotřebí načíst kernel moduly:

sudo modprobe vboxdrv
sudo modprobe vboxnetadp
sudo modprobe vboxnetflt

A to je vše. VirtualBox máte nyní nainstalovaný na svém Arch Linuxu.


Instalace LAMP na Debianu 12

Návod na instalaci webového a databázového serveru s PHP na Debianu 12.

1. Aktualizace systému

Nejprve je třeba aktualizovat náš systém, aby byly všechny balíčky aktuální:

sudo apt update

2. Instalace Apache2

Nyní nainstalujeme webový server Apache2:

sudo apt install apache2

3. Instalace MariaDB

Nyní nainstalujeme databázový server MariaDB:

sudo apt install mariadb-server

Po instalaci MariaDB spustíme interaktivní skript na donastavení databázového serveru:

sudo mysql_secure_installation

Systém se nás nyní zeptá na database root password, tento dotaz stačí odkliknout klávesou ENTER, jelikož jsme MariaDB právě nainstalovali. Na další dvě otázky stačí odpovědět pouze písmenem n. Na další otázky odpovíme písmenem Y, abychom nastavili MariaDB pro produkční prostředí.

4. Instalace PHP

Nyní nainstalujeme PHP:

sudo apt install php libapache2-mod-php php-mysql

A máme hotovo. Adresář webových stránek je implicitně v /var/www/html. Obsah se poté zobrazuje při zadání IP adresy zařízení do Vašeho prohlížeče.
